L: local -- names assigned in any way within a function (def or lambda), and not declared global in that function
E: Enclosing function locals -- Names in the local scope of any and all enclosing functions (def or lambda), from inner outer
G: Global(module) -- Names assigned at top-level of a module file, or decalred global in def within the file
B: Built-in(Pyhton) -- Names preassigned in the built-in names module: open, range, SyntaxError,....
